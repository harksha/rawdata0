** Project setup **
===================

** Installing css processor **
	1. Install node js and restart computer after installing it (unless you already have it)
	2. Open cmd/terminal and run "npm -g install pleeease-cli@3.2.8"
	3. Close and restart cmd/terminal
	
** Using processor **
	1. Open cmd/terminal and navigate to "your-projects-folder\umbraco-nordisk-kulturfond\UmbracoNkf\css";
	2. Run "pleeease watch" - to start the watcher which will recompile the css after every change detected
	OR "pleeease compile" - to compile only once the css
